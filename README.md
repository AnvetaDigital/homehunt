# 🏠 HomeHunt

A modern real estate listing & property finder platform where users can explore, list, and manage properties with an interactive map experience.

## 🚀 Features
- 🔐 Authentication (Buyer/Seller)
- 🏡 Property Listings (Create, Edit, Delete)
- 🔍 Advanced Search & Filters
- 🗺️ Map-based property discovery (Leaflet.js)
- 🖼️ Image uploads (Cloudinary)
- ❤️ Favorites / Saved Properties

## 🛠️ Tech Stack
- Next.js (App Router)
- MongoDB Atlas
- Leaflet.js
- Cloudinary
- Tailwind CSS

## 📂 Project Setup

```bash
git clone https://github.com/AnvetaDigital/homehunt.git
cd homehunt
npm install
npm run dev
